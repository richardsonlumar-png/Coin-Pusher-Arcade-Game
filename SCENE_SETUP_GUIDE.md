# Scene Setup Guide - Coin Pusher Arcade Game\n\n## Overview\nThis guide walks you through creating all game scenes in Unity step-by-step.\n\n## Scenes to Create\n1. **SplashScreen** - Loading screen\n2. **MainMenu** - Game start screen\n3. **GameplayScene** - Main game loop (all 5 machines use this)\n4. **Shop** - Currency & upgrades\n5. **Settings** - Audio/graphics options\n\n---\n\n## Step 1: Setup Project Hierarchy\n\n### Create Folder Structure\nIn your `Assets` folder, create:\n```\nAssets/\n├── Scenes/\n│   ├── SplashScreen.unity\n│   ├── MainMenu.unity\n│   ├── GameplayScene.unity\n│   ├── Shop.unity\n│   └── Settings.unity\n├── Prefabs/\n│   ├── UI/\n│   ├── Coins/\n│   └── VFX/\n├── Materials/\n├── Textures/\n├── Audio/\n│   ├── SFX/\n│   └── Music/\n└── Resources/\n```\n\n---\n\n## Step 2: Create SplashScreen Scene\n\n### 2.1 Create New Scene\n- File → New Scene → **Basic (Built-in)**\n- Save: `Assets/Scenes/SplashScreen.unity`\n\n### 2.2 Setup Camera\n- Delete default camera if exists\n- Create new GameObject: **UI → Panel**\n- Rename to `Canvas`\n- Scene will auto-create Camera & EventSystem\n\n### 2.3 Add Splash Content\nUnder Canvas, create:\n- **Image** → Rename to `Logo`\n  - Set size: 400x400\n  - Center on screen\n  - Add placeholder color\n  \n- **Text (TextMeshPro)** → Rename to `LoadingText`\n  - Text: \"Loading...\"\n  - Position: Center-bottom\n  - Font size: 36\n\n### 2.4 Add Loading Animation Script\nCreate new script: `Assets/Scripts/UI/SplashScreenManager.cs`\n```csharp\nusing UnityEngine;\nusing UnityEngine.SceneManagement;\nusing System.Collections;\n\npublic class SplashScreenManager : MonoBehaviour\n{\n    [SerializeField] private float displayDuration = 3f;\n    [SerializeField] private string nextSceneName = \"MainMenu\";\n    \n    private void Start()\n    {\n        StartCoroutine(ShowSplashScreen());\n    }\n    \n    private IEnumerator ShowSplashScreen()\n    {\n        yield return new WaitForSeconds(displayDuration);\n        SceneManager.LoadScene(nextSceneName);\n    }\n}\n```\n\n**Attach script** to Canvas GameObject\n\n---\n\n## Step 3: Create MainMenu Scene\n\n### 3.1 Create New Scene\n- File → New Scene → **Basic (Built-in)**\n- Save: `Assets/Scenes/MainMenu.unity`\n\n### 3.2 Setup Background\n- Create UI → Panel\n- Rename to `Canvas`\n- Add Image component\n- Set color to dark blue (or add background texture)\n- Stretch to fill: Layout Element → Preferred Width: 1920, Height: 1080\n\n### 3.3 Add Menu Buttons\nUnder Canvas, create these UI elements:\n\n**1. Title**\n- UI → Text (TextMeshPro)\n- Rename to `GameTitle`\n- Text: \"COIN PUSHER\"\n- Font Size: 80\n- Alignment: Center\n- Position: Top-center (Y: 200)\n- Color: Gold/Yellow\n\n**2. Play Button**\n- UI → Button - TextMeshPro\n- Rename to `PlayButton`\n- Text: \"PLAY\"\n- Position: Center (Y: 0)\n- Size: 300x100\n- Add padding\n\n**3. Shop Button**\n- Duplicate PlayButton\n- Rename to `ShopButton`\n- Text: \"SHOP\"\n- Position: Y: -150\n\n**4. Settings Button**\n- Duplicate PlayButton\n- Rename to `SettingsButton`\n- Text: \"SETTINGS\"\n- Position: Y: -300\n\n**5. Daily Reward Display**\n- UI → Panel\n- Rename to `DailyRewardPanel`\n- Add Text for \"Daily Reward: +500 coins\"\n- Position: Top-right\n- Size: 300x100\n\n### 3.4 Button Navigation Script\nCreate: `Assets/Scripts/UI/MainMenuController.cs`\n```csharp\nusing UnityEngine;\nusing UnityEngine.SceneManagement;\nusing UnityEngine.UI;\n\npublic class MainMenuController : MonoBehaviour\n{\n    [SerializeField] private Button playButton;\n    [SerializeField] private Button shopButton;\n    [SerializeField] private Button settingsButton;\n    [SerializeField] private Text dailyRewardText;\n    \n    private void Start()\n    {\n        playButton.onClick.AddListener(OnPlayClicked);\n        shopButton.onClick.AddListener(OnShopClicked);\n        settingsButton.onClick.AddListener(OnSettingsClicked);\n        \n        // Show daily reward\n        if (CurrencyManager.Instance != null)\n        {\n            dailyRewardText.text = \"Daily Reward: +500\";\n        }\n        \n        // Play music\n        AudioManager.Instance?.PlayMusic(\"MainMenu\");\n    }\n    \n    private void OnPlayClicked()\n    {\n        AudioManager.Instance?.PlaySFX(\"button_click\");\n        SceneManager.LoadScene(\"GameplayScene\");\n    }\n    \n    private void OnShopClicked()\n    {\n        AudioManager.Instance?.PlaySFX(\"button_click\");\n        SceneManager.LoadScene(\"Shop\");\n    }\n    \n    private void OnSettingsClicked()\n    {\n        AudioManager.Instance?.PlaySFX(\"button_click\");\n        SceneManager.LoadScene(\"Settings\");\n    }\n}\n```\n\n**Attach to Canvas** and assign button references\n\n---\n\n## Step 4: Create GameplayScene\n\n### 4.1 Create New Scene\n- File → New Scene → **Basic (Built-in)**\n- Save: `Assets/Scenes/GameplayScene.unity`\n\n### 4.2 Setup 3D Environment\n\n**Create Main Camera:**\n- Main Camera position: (0, 2, -5)\n- Look rotation: (10, 0, 0)\n- Field of View: 60\n\n**Create Lighting:**\n- Delete default light\n- Create Directional Light\n  - Rotation: (45, 45, 0)\n  - Intensity: 1.5\n  - Color: White\n\n### 4.3 Create Game Hierarchy\n\nCreate these GameObjects:\n\n**1. GameManager (Empty)**\n- Add components:\n  - GameManager.cs\n  - CurrencyManager.cs\n  - ProgressionManager.cs\n  - CloudSaveManager.cs\n  - AudioManager.cs\n\n**2. PusherPlatform (3D Cube)**\n- Scale: (2, 0.2, 0.5)\n- Position: (0, 0.5, 1)\n- Add PusherController.cs\n- Create child: **SpawnPoint** (position: 0, 2, -1)\n\n**3. PusherBase (3D Cube)**\n- Scale: (3, 0.1, 3)\n- Position: (0, 0, 0)\n- Material: Gray metallic\n- Add BoxCollider\n\n**4. Walls (3D Cubes)**\nCreate left/right walls to contain coins:\n- Left Wall: Position (-2, 0.5, 0.5), Scale (0.2, 2, 4)\n- Right Wall: Position (2, 0.5, 0.5), Scale (0.2, 2, 4)\n- Front Wall: Position (0, 0.5, -2), Scale (4, 2, 0.2)\n\n**5. CoinExit (Trigger)**\n- 3D Cube\n- Scale: (4, 1, 1)\n- Position: (0, -1, 0.5)\n- Check \"Is Trigger\"\n- Add script to detect fallen coins\n\n**6. Canvas (UI)**\n- Add HUD elements:\n  - CoinsText (top-left)\n  - GemsText (top-middle)\n  - JackpotBar (top-right)\n  - SessionStats (bottom-left)\n\n### 4.4 Add HUD Setup\nCreate script: `Assets/Scripts/UI/GameplayHUDSetup.cs`\n```csharp\nusing UnityEngine;\nusing TMPro;\n\npublic class GameplayHUDSetup : MonoBehaviour\n{\n    [SerializeField] private TextMeshProUGUI coinsText;\n    [SerializeField] private TextMeshProUGUI gemsText;\n    [SerializeField] private TextMeshProUGUI xpText;\n    \n    private void OnEnable()\n    {\n        // Subscribe to currency changes\n        if (CurrencyManager.Instance != null)\n        {\n            CurrencyManager.OnCoinsChanged += UpdateCoinsDisplay;\n            CurrencyManager.OnGemsChanged += UpdateGemsDisplay;\n        }\n    }\n    \n    private void UpdateCoinsDisplay(long amount)\n    {\n        coinsText.text = $\"Coins: {FormatNumber(amount)}\";\n    }\n    \n    private void UpdateGemsDisplay(long amount)\n    {\n        gemsText.text = $\"Gems: {FormatNumber(amount)}\";\n    }\n    \n    private string FormatNumber(long num)\n    {\n        if (num >= 1000000) return (num / 1000000f).ToString(\"F1\") + \"M\";\n        if (num >= 1000) return (num / 1000f).ToString(\"F1\") + \"K\";\n        return num.ToString();\n    }\n}\n```\n\n---\n\n## Step 5: Create Shop Scene\n\n### 5.1 Create New Scene\n- File → New Scene → **Basic (Built-in)**\n- Save: `Assets/Scenes/Shop.unity`\n\n### 5.2 Add Shop Layout\n- Create Canvas → Rename to `ShopCanvas`\n- Background Panel (dark overlay)\n\n### 5.3 Add Shop Items\nCreate scrollable list of items:\n- Coin Pack (500 coins)\n- Coin Pack (2500 coins)\n- Gem Pack (50 gems)\n- Premium Theme\n- Ad Removal\n\nCreate script: `Assets/Scripts/UI/ShopController.cs`\n```csharp\nusing UnityEngine;\nusing UnityEngine.UI;\nusing TMPro;\n\npublic class ShopController : MonoBehaviour\n{\n    [SerializeField] private Button backButton;\n    \n    private void Start()\n    {\n        backButton.onClick.AddListener(GoBack);\n    }\n    \n    private void GoBack()\n    {\n        UnityEngine.SceneManagement.SceneManager.LoadScene(\"MainMenu\");\n    }\n}\n```\n\n---\n\n## Step 6: Create Settings Scene\n\n### 6.1 Create New Scene\n- File → New Scene → **Basic (Built-in)**\n- Save: `Assets/Scenes/Settings.unity`\n\n### 6.2 Add Settings Options\n- Master Volume Slider\n- SFX Volume Slider\n- Music Toggle\n- Sound Toggle\n- Graphics Quality\n- Back Button\n\n---\n\n## Step 7: Add Scenes to Build Settings\n\n### Do This IMPORTANT Step:\n1. File → Build Settings\n2. Click \"Add Open Scenes\" for each scene in order:\n   - 0: SplashScreen\n   - 1: MainMenu\n   - 2: GameplayScene\n   - 3: Shop\n   - 4: Settings\n\n---\n\n## Step 8: Test All Scenes\n\n### Test Flow:\n1. Start from SplashScreen\n2. Should auto-load MainMenu after 3 seconds\n3. Click Play → GameplayScene should load\n4. Click Shop → Shop scene\n5. Click Settings → Settings scene\n6. Back buttons return to MainMenu\n\n---\n\n## Next Steps\n\n1. **Add 3D Models** for coins and pusher\n2. **Create Prefabs** for reusable UI elements\n3. **Add Particle Effects** for wins and jackpots\n4. **Import Audio** for background music and SFX\n5. **Test on Mobile Device**\n\n---\n\n## Common Issues & Fixes\n\n### Scenes not loading?\n- Check Build Settings → all scenes listed in order\n- Verify scene names match exactly\n\n### Missing managers?\n- Ensure GameManager has all required components\n- Check DontDestroyOnLoad is set\n\n### UI not showing?\n- Verify Canvas → Render Mode set to Screen Space - Overlay\n- Check all RectTransform sizes\n\n---\n\n**You're all set!** Move to the next step once all scenes are created. ✅\n"
